# GeoX Layout Composer

A professional print layout / report composer subsystem for GeoX that enables users to create publication-ready outputs with viewports, legends, scale bars, north arrows, text annotations, and metadata blocks.

## Features

- **Resolution-independent export** at user-selected DPI (96-600 DPI)
- **Multiple export formats**: PDF, PNG, TIFF
- **Audit trail compliance**: Every export creates a companion `.audit.json` file
- **Deterministic output**: Same project + layout settings = identical output
- **Modular architecture**: Document model separate from UI

## Usage

### Opening the Layout Composer

1. From the main menu: **Reports > Layout Composer...** (Ctrl+L)
2. The Layout Composer window opens with a default layout template

### Quick Export

For fast exports without customization:
- **Reports > Quick Export > Export PDF (300 DPI)...**
- **Reports > Quick Export > Export PNG (300 DPI)...**
- **Reports > Quick Export > Export PNG (600 DPI - Poster)...**

### Creating a Layout

1. **Add Items** using the toolbar or Insert menu:
   - Viewport (3D view capture)
   - Legend
   - Scale Bar
   - North Arrow
   - Text
   - Image/Logo
   - Metadata Block

2. **Position and Resize** items by dragging on the canvas

3. **Configure Properties** using the right panel

4. **Capture Viewport** to update viewport items with current 3D view

5. **Save Layout** as `.geoxlayout.json` template for reuse

### Exporting

1. **Export > Export to PDF/PNG/TIFF...**
2. Select DPI preset or custom resolution
3. Choose output location
4. Export creates:
   - Output file (e.g., `layout.pdf`)
   - Audit record (e.g., `layout.pdf.audit.json`)

## File Format

Layout files use JSON format with `.geoxlayout.json` extension:

```json
{
  "schema_version": "1.0.0",
  "version": "1.0.0",
  "name": "My Layout",
  "page": {
    "size": "A4",
    "orientation": "landscape",
    "width_mm": 297.0,
    "height_mm": 210.0,
    "margin_left_mm": 10.0,
    "margin_right_mm": 10.0,
    "margin_top_mm": 10.0,
    "margin_bottom_mm": 10.0
  },
  "items": [
    {
      "id": "abc12345",
      "item_type": "viewport",
      "name": "Main View",
      "x_mm": 10.0,
      "y_mm": 20.0,
      "width_mm": 150.0,
      "height_mm": 100.0,
      "visible": true,
      "locked": false,
      "z_order": 0,
      "camera_state": { ... },
      "legend_state": { ... }
    }
  ],
  "metadata": {},
  "show_footer": true,
  "footer_text": "Generated by GeoX"
}
```

## Architecture

```
block_model_viewer/
├── layout/                      # Core layout subsystem
│   ├── __init__.py              # Package exports
│   ├── layout_document.py       # Dataclasses: LayoutDocument, PageSpec, Items
│   ├── layout_io.py             # JSON save/load with schema versioning
│   ├── layout_renderer.py       # QPainter-based DPI-independent rendering
│   ├── layout_export.py         # PDF/PNG/TIFF export + audit records
│   └── README.md                # This file
│
└── ui/layout/                   # UI components
    ├── layout_window.py         # Main composer QMainWindow
    ├── layout_canvas.py         # QGraphicsView page canvas
    ├── layout_item_widgets.py   # QGraphicsItems for each item type
    ├── layout_property_panel.py # Property editors
    └── layout_item_list.py      # Item list with visibility/lock
```

## Item Types

| Type | Description | Key Properties |
|------|-------------|----------------|
| `viewport` | 3D view capture | camera_state, legend_state, background_color |
| `legend` | Colormap/category legend | legend_state, orientation, font_size |
| `scale_bar` | Distance reference | units, style, num_segments |
| `north_arrow` | Compass indicator | style, rotation_override |
| `text` | Free text | text, font_family, font_size, alignment |
| `image` | Logo/image | image_path, maintain_aspect, opacity |
| `metadata` | Project info block | fields, show_labels |
